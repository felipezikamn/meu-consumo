# Meu Consumo

Sistema simples para cálculo manual de consumo de energia elétrica.

## Sobre o Projeto

O projeto permite calcular o consumo de energia de aparelhos elétricos de forma simples e rápida, ajudando o usuário a entender melhor o gasto energético e o valor estimado na conta de luz.

## Funcionalidades

- Cálculo de consumo em kWh
- Estimativa de custo mensal
- Interface simples e intuitiva
- Inserção manual dos dados

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Como utilizar

1. Abra o projeto no navegador
2. Insira:
   - potência do aparelho
   - horas de uso
   - dias de utilização
   - valor da tarifa
3. O sistema calculará automaticamente o consumo e o custo estimado.

## Objetivo

O projeto foi desenvolvido com fins acadêmicos e educacionais para auxiliar no entendimento do consumo de energia elétrica.
